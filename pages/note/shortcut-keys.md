# 我的常用快捷键



### 重命名

> 重命名变量：Alt + N



## 注释

> 切换注释	：Ctrl + /



## 格式化代码

> Ctrl + Alt + L



## 行

### 复制

复制当前行至**Clipboard**

> 如果不冲突：Ctrl + C



### 粘贴

粘帖**Clipboard**到当前行

> 如果不冲突：Ctrl + V



### 剪切

剪切当前行

> 如果不冲突：Ctrl + X



### 上移

当前行向上移动

> Alt + UP



### 下移

当前行向下移动

> Alt + DOWN
